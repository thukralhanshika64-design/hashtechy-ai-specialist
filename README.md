# 🪑 Hashtechy AI Specialist — Skill Assessment Submission

**Submitted by:** Hanshika Thukral · **Date:** April 2026

---

This repository contains my submission for the Hashtechy AI Specialist practical assessment. All three tasks were completed independently within the 72-hour window, without any briefing call.

The work spans prompt engineering for a live retail chatbot client, conversational design for a real-time voice AI agent, and building an LLM evaluation framework from scratch — each grounded in production thinking rather than theoretical best practices.

## 📁 What's in this repo

- `Task-S1-EzeeChatBot.md` — Production system prompt, contextual greetings, test set & failure modes for Durian's chatbot
- `Task-S2-VoiceAgent.md` — Full conversational brain for RapidSales.ai's solar outbound voice agent  
- `Task-S3-EvalFramework.md` — 9-metric LLM evaluation framework with drift detection across email, WhatsApp & voice

## 💡 A note on approach

Each task is written to be immediately usable — not a concept document. The system prompts are copy-paste ready, the objection scripts are written as real dialogue, and the evaluation metrics are all objectively measurable without requiring a human to read every conversation. Where I've made deliberate design choices, I've explained the reasoning inline.
